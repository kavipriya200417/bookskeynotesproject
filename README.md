# 📚 BookSky

BookSky is a simple web application that allows users to add, view, and delete books. It is built using HTML, CSS, and JavaScript, focusing on DOM manipulation and basic UI interactions.

---

## 🚀 Features

- Add new books with title, author, and description
- Display books in a clean card layout
- Delete books dynamically
- Popup form for adding books
- Responsive and simple UI design

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## ⚙️ How It Works

- Clicking the **"+" button** opens a popup form
- Users can enter:
  - Book title
  - Author name
  - Description
- Clicking **Add** creates a new book card dynamically using JavaScript
- Clicking **Delete** removes the book from the UI
